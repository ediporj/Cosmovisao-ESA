# Cosmovisão ESA

**ESA** são as iniciais do nome do autor — um ser humano inquieto que, em vez de guardar suas perguntas filosóficas espalhadas por conversas, cadernos e abas mentais abertas, resolveu tentar organizá-las num único lugar.

Este repositório é essa tentativa. Não é uma religião, não é um manifesto, não é doutrina. É o mapa de sentido de uma pessoa específica — feito público porque mapas pessoais às vezes ajudam outros a se orientar.

## O que é isto

A Cosmovisão ESA é uma tentativa de unificar, num único sistema coerente, crenças que estavam fragmentadas em "abas" mentais separadas: catolicismo, espiritismo kardecista, budismo, física moderna, tradições orientais. É um **mapa pessoal de sentido** — em expansão contínua.

## Do triângulo ao círculo

<p align="center">
  <img src="preview.svg" width="520" alt="Animação Cosmovisão ESA"/>
</p>

O framework parte de três vértices — **Alfa, Beta, Gama** — que formam um triângulo inicial. À medida que novos temas são debatidos entre os vértices, surgem pontos intermediários nomeados (αβ, βγ, αγ, αβ₁, αβ₂...). O triângulo se subdivide progressivamente em direção a um círculo. Os arcos em laranja mostram o espaço que ainda separa o polígono do círculo — a distância que permanece entre o que sabemos e o que é.

## Como contribuir

Este framework nasceu de conversas — com pessoas, com textos, com o silêncio. E ainda cresce assim.

Se algo aqui ressoou em você, discordou de você, ou abriu uma pergunta que não tinha antes — isso já é uma contribuição possível. Você pode:

- **Abrir uma Issue** com uma pergunta, objeção ou tensão que o framework ainda não responde bem
- **Propor um novo ponto** do framework — um tema que pertence ao triângulo mas ainda não foi nomeado
- **Sugerir uma resposta** para o Oráculo Walter Aio — se você tem uma forma de dizer algo que o framework já pensa, mas melhor
- **Traduzir** — o framework está em português, mas as perguntas que ele tenta responder não têm língua

Não é preciso concordar com tudo para contribuir. Na verdade, quem discorda com precisão contribui mais do que quem concorda por inércia.

> *"Tudo aqui é hipótese funcional — vale pelo quanto explica e ressoa, não por autoridade."*
